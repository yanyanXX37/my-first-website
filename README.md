# My First HTML Website

A clean, multi-page website built from scratch to master the fundamentals of web development.

[View Live Demo](https://yanyanxx37.github.io/my-first-website/)

## Features
* Hand-coded HTML structure
* Custom CSS styling (No frameworks)
* Responsive design for mobile and desktop
* Linked sub-pages for "About" and "Gallery"

## Tech Used
* HTML5
* CSS3

## How to Run
1. Clone this repository.
2. Open `index.html` in any web browser.
